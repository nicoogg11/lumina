\*\*Lúmina\*\* es una aplicación web moderna diseñada para transformar tu forma de estudiar. Utiliza Inteligencia Artificial (con soporte para Gemini, OpenAI, Claude y modelos locales como Ollama o LM Studio) para convertir tus apuntes, resúmenes y documentos PDF en herramientas de aprendizaje activo basadas en neuroeducación y ciencia cognitiva.



\---



\## ✨ Características Principales



\### 🧠 Métodos de Estudio Basados en Evidencia

Lúmina integra 12 metodologías de estudio activas y estructuradas, generando contenidos personalizados a partir de tus propios materiales:

1\. \*\*Mapa Mental (`datosMapa`)\*\*: Conecta ideas principales, matices y relaciones causa-efecto en un objeto anidado.

2\. \*\*Apuntes Cornell (`datosCornell`)\*\*: Estructura la información en hojas con ideas clave, notas resaltadas (`<b>`) y resúmenes integradores.

3\. \*\*Método Feynman (`FEYNMAN\_DATA`)\*\*: Explica conceptos complejos de forma sencilla, identificando lagunas y puntos críticos de estudio.

4\. \*\*Active Recall (`preguntasRecall`)\*\*: Preguntas de autoevaluación libre para estimular la recuperación de memoria sin opciones.

5\. \*\*Opción Múltiple (`PREGUNTAS`)\*\*: Exámenes de práctica con 4 opciones plausibles y explicaciones conceptuales.

6\. \*\*Repetición Espaciada (`bancoPreguntas`)\*\*: Banco de preguntas y respuestas textuales para repasar justo a tiempo.

7\. \*\*Blurting (`datosBlurting`)\*\*: Vaciamiento de memoria y autoevaluación mediante conceptos clave indispensables.

8\. \*\*Síntesis por Embudo (`datosSintesis`)\*\*: Reducción progresiva del texto en capas de abstracción hasta llegar al \*Elevator Pitch\*.

9\. \*\*SQ3R (`datosSQ3R`)\*\*: Survey, Question, Read, Recite, Review para una lectura activa y profunda.

10\. \*\*Subrayado Jerárquico (`datosSubrayado`)\*\*: Jerarquiza las ideas de un texto mediante categorías codificadas (conceptos, datos, causas, ejemplos, palabras clave).

11\. \*\*Línea de Tiempo (`datosTimeline`)\*\*: Secuencia cronológica conectada para comprender procesos históricos y de causa-efecto.

12\. \*\*Tabla Comparativa (`datosTabla`)\*\*: Contraste estructurado de entidades y criterios lado a lado.



\### ⏱️ Ciclos de Enfoque (Pomodoro) y Gamificación

\- \*\*Temporizador Personalizable\*\*: Configura tus tiempos de enfoque, descansos cortos y descansos largos.

\- \*\*Detección de Abandono\*\*: El temporizador se pausa automáticamente si sales de la pestaña por más de 20 segundos para fomentar la concentración real.

\- \*\*Tu Bosque y Rachas\*\*: Cada sesión de enfoque completada planta un árbol en tu bosque virtual y aumenta tu racha diaria de estudio.

\- \*\*To-Do List Integrado con XP\*\*: Gestiona tus tareas con prioridades (Normal, Media, Alta), usa plantillas predefinidas para exámenes o entregas, gana puntos de experiencia (XP) y sube de nivel.



\### 🤖 Multi-Proveedor de Inteligencia Artificial

\- \*\*Gemini\*\*: Modelos rápidos y potentes de Google (Flash y Pro).

\- \*\*ChatGPT\*\*: Modelos GPT de OpenAI.

\- \*\*Claude\*\*: Modelos de Anthropic.

\- \*\*IA Local (OpenAI Compatible)\*\*: Conéctate a servidores locales como \*\*LM Studio\*\* u \*\*Ollama\*\* para procesar tus documentos de forma 100% privada y offline.

\- \*\*Streaming en Tiempo Real\*\*: Visualiza la respuesta de la IA a medida que se genera.

\## 🛠️ Tecnologías Utilizadas



\- \*\*React 18\*\*: Interfaz de usuario interactiva basada en componentes.

\## 🚀 Instalación y Ejecución Local



Sigue estos pasos para clonar y ejecutar Lúmina en tu máquina:



1\. \*\*Clona el repositorio\*\*:

&#x20;  ```bash

&#x20;  git clone https://github.com/tu-usuario/lumina-estudio.git

&#x20;  cd lumina-estudio

&#x20;  ```



2\. \*\*Instala las dependencias\*\*:

&#x20;  ```bash

&#x20;  npm install

&#x20;  ```



3\. \*\*Inicia el servidor de desarrollo\*\*:

&#x20;  ```bash

&#x20;  npm run dev

&#x20;  ```



4\. Abre tu navegador en `http://localhost:5173`.



\---



\## 💡 Cómo Usar Lúmina



1\. \*\*Conecta tu IA\*\*: Haz clic en el botón superior derecho \*\*"Conectar IA"\*\* e introduce tu API Key o configura la URL de tu servidor local (ej. `http://localhost:1234` para LM Studio).

2\. \*\*Crea una Materia\*\*: Organiza tus documentos por carpetas en la sección \*\*Biblioteca\*\*.

3\. \*\*Sube tu Material\*\*: Agrega archivos en formato `.pdf`, `.txt` o `.md`.

4\. \*\*Estudia Activamente\*\*: Elige cualquiera de los 12 métodos en la sección de inicio o haz clic en "Comenzar a estudiar" para entrar a la \*\*Sala de Estudio\*\* inmersiva.
